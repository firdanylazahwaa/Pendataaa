## Memahami Data 

**Pengertian Data Understanding**

Data Understanding adalah tahap awal dalam proses data mining yang bertujuan untuk mengumpulkan, mengenali, dan mempelajari data sebelum dilakukan pengolahan lebih lanjut.

Pada tahap ini, analis mulai berinteraksi langsung dengan data untuk memahami:

* isi data

* struktur data

* kualitas data

* potensi permasalahan data

Tahap ini sangat penting karena kesalahan memahami data dapat menyebabkan hasil analisis tidak akurat.

**Aktivitas dalam Data Understanding**

**1. Collect Initial Data**

Tahap mengumpulkan data dari berbagai sumber.

Kegiatan:

* Mengambil data dari file, database, atau API

* Memastikan data relevan dengan tujuan

* Mengetahui jumlah data

* Membuat laporan data awal

Contoh:
Mengambil dataset Iris dari file CSV.

**2. Describe Data**

Tahap mendeskripsikan karakteristik data.

Yang dilakukan:

* Mengetahui jumlah baris dan kolom

* Mengidentifikasi tipe data

* Memahami arti setiap atribut

* Melihat struktur dataset

Contoh pada dataset Iris:

* 150 baris data

* 5 kolom

* 4 numerik + 1 kategorikal

**3. Explore Data**

Tahap eksplorasi untuk menemukan pola awal.

Kegiatan:

* Membuat visualisasi

* Melihat distribusi data

* Mencari hubungan antar variabel

* Mengidentifikasi outlier

Tools yang sering dipakai:

* Scatter Plot

* Box Plot

* Correlation

* Rank
