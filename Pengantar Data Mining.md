## Pengantar Data Mining

**Pengertian Data Mining**

Data mining adalah proses mengekstraksi pola, hubungan, atau informasi penting dari kumpulan data berukuran besar menggunakan teknik statistik, machine learning, dan basis data.

Proses ini bertujuan mengubah data mentah menjadi pengetahuan yang bermanfaat untuk mendukung pengambilan keputusan.

**Tujuan Data Mining**
Tujuan utama data mining meliputi:

* Menemukan pola tersembunyi dalam data.

* Melakukan prediksi berdasarkan data historis.

* Mengelompokkan data yang memiliki karakteristik mirip.

* Mengidentifikasi anomali atau penyimpangan data.

* Mendukung pengambilan keputusan berbasis data.


**Manfaat Data Mining**

Beberapa manfaat data mining dalam berbagai bidang:

* Bisnis: memahami perilaku pelanggan.

* Perbankan: deteksi penipuan (fraud).

* Kesehatan: prediksi penyakit.

* Pendidikan: analisis performa mahasiswa.

* E-commerce: sistem rekomendasi produk.

**Tahapan Umum Data Mining**

Secara umum, proses data mining mengikuti metode CRISP-DM, yaitu:

* Business Understanding


* Data Understanding

* Data Preparation

* Modeling

* Evaluation

* Deployment

Pada tugas ini, fokus utama berada pada tahap Data Understanding.
